# ECMAScript Modules (ESM) Support

**Category**: Interoperability
**Status**: Completed
**Difficulty**: High
The parser and compiler have been heavily upgraded to support standard `import`/`export` syntax. Curica natively resolves asynchronous module graphs, fetches remote HTTP imports, and safely compiles Top-Level Await instructions, bringing modern, standard web module architecture to the backend without relying on bundlers.
