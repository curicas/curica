# Built-in Toolchain

**Category**: Build_and_Tooling
**Status**: Completed
**Difficulty**: Low
Taking inspiration from modern runtimes, Curica is integrating a native C-based toolchain directly into the executable. This includes an ultra-fast code formatter and a zero-overhead TypeScript type stripper. This allows developers to execute `.ts` files immediately without installing `node_modules` or running a separate `tsc` build step.
